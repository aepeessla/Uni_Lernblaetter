Inonen

Wie entsteht eine Ionenbindung? (z. B. bei Kochsalz)
  • Durch die vollständige Übertragung v. Elektronen.
  • Ein Atom (Natrium) $\underrightarrow{\ \ \ \ \textcolor{#c72483}{\text{gibt ab}}\ \ \ \ }$ Elektron $\underrightarrow{\ \ \ \ \textcolor{#c72483}{\text{wird zum}}\ \ \ \ }$ positiven Kation ($Na^+$), d. andere (Chlor) nimmt es auf $\underrightarrow{\ \ \ \ \textcolor{#c72483}{\text{wird zum}}\ \ \ \ }$ negativen Anion ($Cl^-$). Die entgegengesetzten Ladungen ziehen sich elektrostatisch an.
In welcher Struktur liegen die Ionen im festen Zustand vor?
  • bilden ein festes, regelmäßiges Kristallgitter (Ionengitter)
Welche Voraussetzung muss erfüllt sein, damit sich ein Salzkristall in Wasser auflöst?
  • $\text{Hydratationsenergie} > \text{Gitterenergie}$
Was versteht man unter der "Gitterenergie" ?
  • Energie, d. Ionen im festen Kristalgitter zsm.hält $\underrightarrow{\ \ \ \ \textcolor{#c72483}{\text{Überwindung}}\ \ \ \ }$ Zerstörung des Kristalgitters
Was versteht man unter der "Hydratationsenergie" ?
  • Energie, d. freigesetzt wird, wenn sich Wassermoleküle an d. freigewordenen Ionen anlagern.
Was versteht man unter der "Hydratationsenergie" ?
  • Eine Schicht aus Wassermolekülen, d. sich beim Lösen um d. einzelnen Ionen herum anordnet.
  • Dabei zeigt d. positive Pol des Wassers ($H$) zum negativen Chloridion ($Cl^-$) & d. negative Pol ($O$) zum positiven Natriumion ($Na^+$).

Wasserstoffbrückenbindung

Welcher Prozess beschreibt den ersten Schritt beim Lösen eines Salzes (aus energetischer Sicht) ?
  • D. Aufbrechen des Ionengitters. Die Ionen müssen aus ihrem festen Verband gelöst werden (dafür wird Gitterenergie verbraucht).
Was ist die Gitterenergie ($U_L$)?
  • feste Kristallgitter $\underrightarrow{\ \ \ \ \textcolor{#c72483}{\text{GItterenergie}}\ \ \ \ }$ gasförmige Ionen
  • immer positiv (endotherm).
Was ist die Gitterenergie ($U_L$)?
  • feste Kristallgitter $\underrightarrow{\ \ \ \ \textcolor{#c72483}{\text{GItterenergie}}\ \ \ \ }$ gasförmige Ionen
  • immer positiv (endotherm).
Welcher Prozess beschreibt den zweiten Schritt beim Lösen ?
  • Hydratation: Ionen $\underrightarrow{\ \ \ \ \textcolor{#c72483}{\text{umgeben}}\ \ \ \ }$ Wassermoleküle $\underrightarrow{\ \ \ \ \textcolor{#c72483}{\text{Entstehung}}\ \ \ \ }$ Hydrathülle $\underrightarrow{\ \ \ \ \textcolor{#c72483}{\text{wird frei}}\ \ \ \ }$ Hydratationsenergie
Was bestimmt, ob sich ein Salz löst oder nicht ?
  • Die Lösungsenthalpie ($\Delta H_{sol}$): $Gitterenergie + Hydratationsenergie$
Wie lautet die Faustformel für die Lösungsenthalpie?
  • $\Delta H_{sol} = U_L + \Delta H_{hyd}$
Was passiert energetisch, wenn sich ein Salz löst (exotherm vs. endotherm)?
  1. Exotherm: Es wird mehr Energie bei der Hydratation frei, als für das Gitter benötigt wurde ($\Delta H_{sol} < 0$) $\implies$ Glas wird warm.
  2. Endotherm: Es wird mehr Energie benötigt, um das Gitter zu knacken, als bei d. Hydratation frei wird ($\Delta H_{sol} > 0$) $\implies$ Glas kühlt ab.
