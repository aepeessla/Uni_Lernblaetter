Normalenformen

Was ist eine Normalform ?

  • allgemeine Form = $ax^2 + 2bxy + cy^2 + dx + ey + f = 0$
  • Durch Drehung d. Koordinate wird d. Term xy eliminiert $\underrightarrow{\ \ \ \ \textcolor{#83b7ea}{\text{Normalform sieht dann so aus}}\ \ \ \ }$ $\lambda_1 u^2 + \lambda_2 v^2 + g = 0$
    • $\lambda_1$ & $\lambda_2$ = Eigenwerte d. Matrix $\underrightarrow{\ \ \ \ \textcolor{#83b7ea}{\text{repräsentiert}}\ \ \ \ }$ quadratischen Teil d. Gleichung
Wie Normalform bestimmen ?
  • 3 Schritte

    1. Matrix aufstellen

      $\begin{pmatrix} a & b \ b & c \end{pmatrix}$

    2. Eigenwerte bestimmen

    3. Einsetzten in d. Normalform

λ1 λ2 h Typ Beispiel
+ + + leere Menge u² + v² + 1 = 0
+ + 0 Punkt u² + v² = 0
+ + - Ellipse u² + v² = 1
+ 0 + leere Menge u² + 1 = 0
+ 0 0 Gerade (Doppelgerade) u² = 0
+ 0 - zwei parallele Geraden u² - 1 = 0
+ - + Hyperbel u² - v² = -1
+ - 0 zwei sich schneidende Geraden u² - v² = 0
+ - - Hyperbel u² - v² = 1
Bezeichne die einzelnen Bestandteile: $\begin{pmatrix} 1 \\ 0 \\ 2 \end{pmatrix} + s \cdot \begin{pmatrix} 2 \\ 1 \\ 0 \end{pmatrix} + t \cdot \begin{pmatrix} 0 \\ 1 \\ 3 \end{pmatrix}$

  • $\begin{pmatrix} 1 \ 0 \ 2 \end{pmatrix}$ = Stützvektor

  • $\begin{pmatrix} 2 \ 1 \ 0 \end{pmatrix}$ & $\begin{pmatrix} 0 \ 1 \ 3 \end{pmatrix}$ = Spannvektoren

Was muss ich verw., wenn ich zeigen möchte, ob 2 Vektoren orthogonal zu einander sind ?
  • Skalarprodukt
    • $<a,b> = 0$
Wie lautet d. Formel zur Berechnung v. Winkeln ?
  • $\frac{<a,b>}{|a| \cdot |b|}$
Mit was kann ich 2 Vektoren fussieren ?
  • Vektorprodukt
    • $|a \times b| = \begin{pmatrix} a_1 \ a_2 \ a_3 \end{pmatrix} \times \begin{pmatrix} b_1 \ b_2 \ b_3 \end{pmatrix}$
Was ist der Normalvektor ?
  • Vektorprodukt
    • $|a \times b| = \begin{pmatrix} a_1 \ a_2 \ a_3 \end{pmatrix} \times \begin{pmatrix} b_1 \ b_2 \ b_3 \end{pmatrix}$
Wie bestimmt man die Normalform von Ebenen ?

  • $E: (\vec{x} - \vec{c}) \cdot (\vec{a} \times \vec{b}) = 0$

  • $[x - \text{Stütztvektor}] \cdot \text{Normalvektor} = 0$

    Was bedeutet es, wenn das Ergebnis $E \ne 0$ ist?
    • Punkt liegt $\lnot$ auf d. Ebene
Gegeben ist $A = \begin{pmatrix} 2 \\ 1 \\ 3 \end{pmatrix}, \quad B = \begin{pmatrix} 4 \\ 1 \\ 4 \end{pmatrix}, \quad C = \begin{pmatrix} 2 \\ 3 \\ 5 \end{pmatrix}$. Wie bestimmt man die Spannvektor ?
  • u = AB = B-A
  • v = AC = C-A
Gegeben ist $A = \begin{pmatrix} 2 \\ 1 \\ 3 \end{pmatrix}, \quad B = \begin{pmatrix} 4 \\ 1 \\ 4 \end{pmatrix}, \quad C = \begin{pmatrix} 2 \\ 3 \\ 5 \end{pmatrix}$. Bestimme d. Ebenengleichung ?

Wie berechne ich die parametrische Form von Ebenen ?

$\boxed{x_1 + x_2 + x_3 = 4}$

  1. Punkte finden
    • $A(4, 0, 0)$ (für $x_1=4, x_2=0, x_3=0$)
    • $B(0, 4, 0)$ (für $x_1=0, x_2=4, x_3=0$)
    • $C(0, 0, 4)$ (für $x_1=0, x_2=0, x_3=4$)
  2. Vektoren bilden
  • Richtungsvektoren berechnen:

    • $u = AB = B - A= \begin{pmatrix} 0-4 \ 4-0 \ 0-0 \end{pmatrix} = \begin{pmatrix} -4 \ 4 \ 0 \end{pmatrix}$

    • $v = AC = C - A= \begin{pmatrix} 0-4 \ 0-0 \ 4-0 \end{pmatrix} = \begin{pmatrix} -4 \ 0 \ 4 \end{pmatrix}$

  1. Parameterform:
    • $\vec{x} = \begin{pmatrix} 4 \ 0 \ 0 \end{pmatrix} + r \cdot \begin{pmatrix} -4 \ 4 \ 0 \end{pmatrix} + s \cdot \begin{pmatrix} -4 \ 0 \ 4 \end{pmatrix} \quad (r, s \in \mathbb{R})$

Schnittpunkte

Wie berechnet man den Schnittpunkt 2er Ebenen ?
  1. Gleichsetzen mit Addition oder Subtraktion
  2. Nach Var. umstellen
  • wenn Gleichungen $\lt$ Variabeln => ein Var. frei wählbar
  1. Parameterform aufschreiben
Wie berechnet man den Schnittpunkt von einer Gerade und einer Ebene ?

wenn es geschirben ist als $(1,2,3) + \lambda(1,2,3)$, dann ist es eigentl.: $\begin{pmatrix} 1 \ 2 \ 3 \end{pmatrix} + \lambda \cdot \begin{pmatrix} 1 \ 2 \ 3 \end{pmatrix}$

  1. Gerade in Einzelteile zerlegen
  2. Einzelteile in Ebene einsetzen
  3. N. \lambda umstellen
  4. In Formel eins.: $\boxed{a + \lambda \cdot b}$

Unterraum

Was muss gelten, sodass eine Gerade ein Unterraum ist ?
  • Es muss immer d. Nullpunkt des Koordinatensystems im Unterraum enthalten sein
    • wenn Ebene & Gerade $\lnot$ durch den Punkt $\implies$ autom. $\lnot$ im Unterraum
  1. Nullvektor enthalten ?
  2. Abgeschlossenheit durch Addition: U + U = U
  3. Abgeschlossenheit durch Multiplikation: Vektorraum des Unterraum aus U \cdot \lambda = immer in U ?
Wie bestimme ich d. Basen im Unterraum ?

  • Ich muss d. parametrische Form v. U darstellen & d. 2 Vektoren sind dann autom. meine Basen

  • Bsp: $$U := {(x_1,x_2,x_3) \in \mathbb{R}^3: x_1 + x_2 + x_3 = 0}$$ 1 Gleichung $<$ 3 Var. = 2 Var. frei $\implies x_2 = s, x_3 = t$ $$x_1+t+s = 0$$ $$x_1 = -t-s$$ $$U = \begin{pmatrix} -t-s \ t \ s \end{pmatrix} = t \cdot \begin{pmatrix} -1 \ 1 \ 0 \end{pmatrix} + s \cdot \begin{pmatrix} -1 \ 0 \ 1 \end{pmatrix}$$ Somit ist die Basis autom.: $B = \left{ \begin{pmatrix} -1 \ 1 \ 0 \end{pmatrix}, \begin{pmatrix} -1 \ 0 \ 1 \end{pmatrix} \right}$
